# 🚀 로컬 개발 환경 구동 가이드

이 프로젝트는 **Frontend(React)**와 **Backend(Spring Boot)** 서버를 각각 실행해야 정상적으로 작동합니다. 아래 순서에 따라 서버를 구동해 주세요.

---

## 1. 전제 조건
- **Node.js**: 18 버전 이상
- **Java**: JDK 17 버전 이상

---

## 2. 서버 구동 순서

### ⚙️ 단계 1: Backend 서버 실행 (Spring Boot)
데이터베이스 처리 및 API 제공을 위한 서버를 먼저 실행합니다.

1. 터미널을 열고 `backend` 디렉토리로 이동합니다.
   ```bash
   cd backend
   ```
2. 서버를 실행합니다.
   ```bash
   ./gradlew bootRun
   ```
   - 서버가 정상적으로 실행되면 `http://localhost:8080` 포트로 대기합니다.

### 🖥️ 단계 2: Frontend 서버 실행 (React)
화면을 구성하고 백엔드와 통신하는 프론트엔드 서버를 실행합니다.

1. **새로운 터미널 창**을 엽니다. (백엔드 서버 터미널은 유지)
2. 프로젝트 루트 디렉토리(`MY_SHOP_FRONT`)에서 다음 명령어를 입력합니다.
   ```bash
   # 필요한 패키지 설치 (처음 한 번만 실행)
   npm install

   # 개발 서버 실행
   npm start
   ```
3. 실행이 완료되면 브라우저에서 `http://localhost:3000`으로 접속됩니다.

---

## 3. 요약 및 확인
- **프론트엔드 서버**: `http://localhost:3000` (UI 화면 확인)
- **백엔드 서버**: `http://localhost:8080/api/v1` (API 데이터 제공)
- 두 서버가 모두 켜져 있어야 로그인이 가능하고 데이터가 화면에 출력됩니다.
